A beautiful, frontend-only task management dashboard designed to help individuals and small teams efficiently organize daily tasks, track progress, and manage priorities without the need for complex backend integrations.

All data is seamlessly and securely managed on the client side using the **Browser localStorage API**, ensuring lightning-fast performance and privacy.

## ğŸš€ Getting Started

Follow these instructions to set up the project locally on your machine.

### Prerequisites

You need [Node.js](https://nodejs.org/) and `npm` installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Vaishakharekere/smart-task-dashboard.git
   cd smart-task-dashboard
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm start
   ```

4. **View the Application:**
   Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
